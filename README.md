API
===
```
TestHelper::getPrivatePropertyValue($classOrObject, $propertyName)
```

```
TestHelper::setPrivateProperty($classOrObject, $propertyName, $propertyValue)
```

```
TestHelper::getPrivateMethod($className, $methodName)
```
